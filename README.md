# EonsPatch

**Public Source MPQ — World of Warcraft 3.3.5a (Build 12340)**

This repository provides all MPQ patch files for the **EonsPatch** project, including base client patches and French (frFR) locale patches. Each file is distributed as an individual GitHub Release for easy downloading.

## 🔧 Installation

1. Download the required `.MPQ` files from the tables above.
2. Place them in your WoW client's `Data/` directory (for base patches) or `Data/frFR/` (for French locale patches).
3. Make sure your client is version **3.3.5a (Build 12340)**.
